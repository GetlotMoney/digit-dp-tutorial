# 8. 模板总结与技巧

## 8.1 通用模板骨架

```cpp
ll dfs(int pos, bool limit, bool lead0, /* 题目额外状态 */) {
    if (pos == -1) return /* 边界返回值 */;
    if (!limit && !lead0 && vis[pos][...]) return dp[pos][...];
    int up = limit ? a[pos] : 9;
    ll res = 0;
    for (int d = 0; d <= up; d++) {
        // 过滤不合法的 d
        res += dfs(pos - 1, limit && d == up, lead0 && d == 0, /* 更新后的额外状态 */);
    }
    if (!limit && !lead0) { vis[pos][...] = true; dp[pos][...] = res; }
    return res;
}
```

## 8.2 常见技巧清单

- **前缀和转化：** 区间问题一律 `f(R) - f(L-1)`，f 定义为 [0,N] 含 0。
- **记忆化范围：** 只缓存 `limit=0 && lead0=0` 的状态（无重复子问题且不依赖 a[pos]）。
- **前导零处理：** `lead0` 影响有效位数和「是否真正出现某数字」，务必单独判断；前导零的 0 不要进入「历史位」状态。
- **求和 vs 计数：** 计数题直接累加；求和（权值）题的贡献要乘以子树计数，dfs 需返回 {个数, 和}。
- **维度压缩：** 模数等循环状态维度用 `% K` 控制，避免数组过大。
- **多询问：** 多组数据记得清空 `vis`，或把 `vis` 改成时间戳。
- **滑动窗口状态：** 峰谷/单调类题用 `pre2, pre1` 两位滑动，比记整个前缀高效。
