# 7. 综合应用与变体

## 7.1 二进制数位 DP

把「10 进制」换成「2 进制」，常用于统计区间内二进制表示中 1 的个数、异或性质等。模板完全相同，只是 `up` 变成 1。

## 7.3 数位 DP + AC 自动机

当条件是「不含若干个子串」且子串很多，可以把子串建 AC 自动机，数位 DP 的额外状态变成「在 AC 自动机上的节点编号」。这是进阶中的进阶。
